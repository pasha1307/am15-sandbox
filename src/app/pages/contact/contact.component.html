Two scenarios to use Smarty Addresses:

1. UPDATE. If the address exists and needs change or verification with the common DB (Smarty)

2. CREATE. If a new address is added that comes from the Smarty DB (complies with Smarty).

Create (Add New) Address:

  • Step 1. Click Add New Button. The modal opens
  • Step 2. Start typing with the building number first. (there is a delay to start searching after 3-4 characters are typed ).
  • Step 3. Chooses the address and fields are populated with data (no apartments )
  • Step 4. If there are more than 0 numbers in (2), then click the second time to select the apartment/unit number.
  • Step 5. Click Submit. The Modal closes and the address is added to the appeallant.

Update (Verify) Address:

  • Step 1. Click the Address you want to update or verify. The modal opens
  • Step 2. Check in the right window whether the address complies with SMARTY.
  • Step 3. If Yes, then click Submit
  • Step 4. If not, then type info in the fields except zip code and click Verify.
  • Step 5. Click Submit. The Address is updated in the Appellant's list.
